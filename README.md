# Personal Portfolio - Applied Mathematics & Computer Science

This repository contains my personal portfolio website, designed to showcase the intersection of mathematical theory and software engineering. It highlights my academic journey at Politehnica University of Bucharest and my technical expertise in Data Science and Machine Learning.

## Project Overview
The portfolio is a single-page application (SPA) built with React, focusing on clean architecture and high scannability for recruiters and collaborators.

## Technical Stack
* **Core:** React.js
* **Styling:** CSS Modules (BEM-inspired naming convention)
* **Data Management:** Local JSON structures for modular content updates
* **Environment:** Vite / Node.js

## Features
* **Interests Section:** Detailed breakdown of my focus in Data Science and Machine Learning, specifically highlighting my Bachelor's thesis on academic performance prediction.
* **Dynamic Skills Grid:** Categorized display of technical proficiencies (Python, C++, SQL, React) and strategic soft skills (Change Management, Project Management).
* **Experience Timeline:** A chronological overview of my professional and academic growth.
* **Projects Gallery:** A curated list of technical projects and publications with external links.
* **Responsive Contact Footer:** Direct integration with professional social platforms and email.
